# Планировщик заданий и дедлайнов для студенческой группы

## О проекте
Веб-приложение для совместного планирования учебных заданий, лабораторных, курсовых и дедлайнов внутри одной группы/курса.

## Цель
Помочь студентам не пропускать дедлайны, видеть общий прогресс группы и упростить координацию.

## Ключевые функции (MVP)
- Добавление/редактирование заданий (предмет, дедлайн, баллы)
- Отметка выполнения каждым студентом
- Общий календарь дедлайнов
- Личный и групповой прогресс
- Напоминания о приближающихся сроках
- Фильтрация заданий

## Технологии (планируемые)
- Backend: Python (FastAPI / Django) / Node.js
- Frontend: React / Vue / чистый HTML+JS+CSS
- База данных: PostgreSQL / SQLite
- Уведомления: в будущем email / push

Инструкция появится после реализации backend и frontend.
